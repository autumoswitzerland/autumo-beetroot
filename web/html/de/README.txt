
Verwenden Sie dieses Verzeichnis, wenn Sie HTML-Vorlagen überschreiben wollen.
Richten Sie dazu die gleiche Verzeichnisstruktur wie unter "web/html/" ein und
benennen die HTML-Templates genau so wie unter "web/html/" und deren
Verzeichnisstruktur.

Dies ist praktisch, wenn Sie neben der Standard-Sprachübersetzung die
HTML-Templates für verschiedene Sprachen unterschiedlich gestalten wollen.

beetRoot lädt zunächst die HTML-Templates, je nach der vom Benutzer gewählten
Sprache, aus solchen vom Benutzer gewählten Sprache, aus solchen
Sprachverzeichnissen und, falls ein solches Verzeichnis nicht vorhanden ist
aus dem Basisverzeichnis "web/html/".
