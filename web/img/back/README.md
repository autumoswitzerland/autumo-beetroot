## Background Fader Removal

If you want to remove the background fader, follow these steps:

1. Delete the `<script>` tag and the `<div>` element with the class `bg-fader` from `web/html/users/login.html`.
2. Remove the entire `:root` section and the `.bg-fader` rules from `web/css/refs.css`.
